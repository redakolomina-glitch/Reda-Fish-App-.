# 🐟 Reda Fish App

**Reda Fish** هو تطبيق لبيع الأسماك في العيون، كيتيح للزبناء يشوفوا جميع أنواع الأسماك، الأسعار ديالهم، ويقدروا يطلبوا التوصيل مباشرة.

![Reda Fish Logo](https://i.imgur.com/J3xZr3r.png)

---

## 🌟 الميزات الرئيسية

- عرض جميع أنواع الأسماك المتوفرة في المتجر
- أسعار محدثة يومياً
- خدمة التوصيل للمنازل
- الاتصال المباشر بالمتجر عبر الهاتف

![Fresh Fish Market](https://i.imgur.com/4lG8y9m.jpg)

---

## 📱 كيفية الاستخدام

1. فتح التطبيق على الهاتف  
2. تصفح أنواع الأسماك  
3. اختيار الأسماك المرغوبة وإضافة إلى الطلب  
4. طلب التوصيل أو الاتصال بالمتجر  

![Fish Delivery](https://i.imgur.com/0FvGdD9.jpg)
